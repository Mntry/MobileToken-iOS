Monetary MobileToken iOS Framework

Securely tokenize user-entered credit card account data!

Use MonetaryMobileToken.framework in your iOS project to prompt users for credit card account data and receive a one-time-use token for secure processing with Monetary's payment APIs. For more information, see our quick-start guide!
